I just spent five hours of my Christmas Monday on installing gfortran on MacOS High Sierra! I was getting the following error while running some simple Fortran codes for earthquake source simulations. I correctly thought it is due to some configuration problems of the latest version of gfortran -that I installed from the GNU project- with recent MacOS. I was correct, it turned out that the latest releases (ver>7) of gfortran (maybe only for this particular use) is not compatible with High Sierra anymore.
FATAL:/opt/local/bin/../libexec/as/x86-64/as: I don't understand 'm' flag!
A solution that I found after hours of struggle, was to install an older version gcc6.4 that with itself has gfortran6. This could be installed via MacPorts:
sudo port install gcc6
and will have the following name in binaries folder: "gfortran-mp-6".
